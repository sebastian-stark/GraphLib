◆ coordinates

std::vector<coordinate_t> GraphLib::Edge::coordinates
private

A vector of (x,y) pairs forming the edge. The first element in the vector corresponds to the "location" of Edge::node_1. The last element in the vector corresponds to the "location" of Edge::node_2. The other elements in the vector describe the "shape" of the edge as one goes from Edge::node_1 to Edge::node_2.
